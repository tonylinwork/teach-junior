# 測驗內容擴充指南

如果您想要加入新的章節或題目，請參考以下步驟：

### 1. 準備 HTML 資源
1. 將新的 Word 檔案另存為 **HTML (網頁)** 格式。
2. 將產生的 `.html` 檔案放入 `src/data/raw/` 目錄中。
3. 將該 HTML 對應的圖片資料夾中的所有圖片，複製到 `public/assets/images_ans/`。

### 方式一：UI 匯入（推薦）
1. 將您的 Word 檔案另存為 **HTML (網頁)** 格式。
2. 在首頁點擊 **「匯入新章節」**。
3. 點擊按鈕並選擇剛才匯出的 **完整資料夾**。
4. 系統會自動填好標題與代碼，點擊 **「開始匯入」** 即可。

### 方式二：刪除章節
1. 在網頁上方章節切換選單右側，點擊 **「垃圾桶」** 圖示。
2. 在彈出的確認視窗點擊「確定」。
3. 系統會自動移除相關的 JSON、HTML 以及圖片資源。

### 方式三：終端機手動轉換（備用）
如果您想手動執行，可以開啟終端機執行解析腳本：

```bash
# 格式：node scripts/parse_quiz.js <輸入路徑> <輸出路徑> <章節名稱> <圖片子目錄>
node scripts/parse_quiz.js src/data/raw/chapter2.html src/data/chapter2.json "第一冊第二章" chapter2
```

### 數學公式支援 (KaTeX)

系統現已支援 **LaTeX** 高品質公式渲染。您可以選擇使用傳統圖片或新的 LaTeX 語法。

#### 1. 使用 LaTeX 公式 (推薦)
在 Word 中或 JSON 裡，使用以下語法：
- **行內公式**： menggunakan `$x^2$` 會融入文字行。
- **區塊公式**：使用 `$$公式$$` 會獨立成行並置中顯示。
- **填答區**：學生可以在填答框直接輸入 LaTeX 語法，系統會提供即時預覽。

#### 2. 使用傳統圖片公式
- 系統會自動解析 Word 匯出的 `<img src="...">`。
- 目前系統採取 **混合模式**：您可以同時在同一個章節中使用 LaTeX 字串與舊有圖片，它們會完美融合顯示。

#### 3. 數學鍵盤
- 點擊「數學鍵盤」按鈕會插入對應的 LaTeX 代碼（例如 `\sqrt{}`）。
- 填答者可以自行修改代碼，預覽區域會即時呈現結果。

---
*提示：如果新的 HTML 結構與第一章差異很大（例如 Class 名稱不同），可能需要微調 `scripts/parse_quiz.js` 中的 `selector` 設定。*
